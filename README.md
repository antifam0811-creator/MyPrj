# Google Colab 마인크래프트 서버 생성기

이 프로젝트는 Google Colab 환경에서 마인크래프트 서버를 구축하고, 설정 파일을 생성하며, 최종 결과물을 압축 파일로 추출할 수 있도록 돕는 자동화 도구입니다.

## 주요 기능

1.  **Java 자동 설치**: 서버 버전에 적합한 JDK(8, 11, 16, 17, 21)를 선택하여 설치할 수 있습니다.
2.  **서버 코어 다운로드**: PaperMC API를 통해 지정된 버전의 서버 파일(.jar)을 내려받으며, EULA(이용약관) 동의 프로세스를 자동화합니다.
3.  **설정 자동화**: server.properties의 주요 항목들을 Colab의 입력 폼을 통해 수정하고 파일로 적용할 수 있습니다.
4.  **결과물 내보내기**: 구축된 서버 폴더 전체를 .zip 형식으로 압축하여 로컬 환경으로 다운로드하는 기능을 제공합니다.

## 사용 방법

1.  저장소 내의 `서버 파일 생성기.ipynb` 파일을 Google Colab에서 실행합니다.
2.  **JAVA INSTALLATION**: 사용하고자 하는 자바 버전을 선택하고 셀을 실행합니다.
3.  **DOWNLOAD SERVER FILE**: 서버 파일 주소를 확인하거나 기본값으로 실행하여 파일을 다운로드합니다.
4.  **SERVER.PROPERTIES**: 게임모드, 난이도, 인원 제한 등 세부 설정을 입력하고 실행하여 설정 파일을 생성합니다.
5.  **START SERVER**: 설정된 메모리 값으로 서버가 정상 구동되는지 확인합니다.
6.  **EXPORT SERVER FOLDER**: 모든 작업이 완료되면 압축 파일 이름을 지정하고 다운로드합니다.

## 주의 사항

  * **보안**: 설정 파일 내에 포함된 관리용 비밀키나 고유 식별 정보가 외부에 노출되지 않도록 주의하십시오.
  * **환경 유지**: Google Colab의 특성상 세션이 종료되면 저장되지 않은 파일은 삭제됩니다. 작업 완료 후 반드시 내보내기 기능을 사용해 파일을 백업하시기 바랍니다.
